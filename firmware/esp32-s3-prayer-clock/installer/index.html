Install Adhan Clock

Flash the standalone prayer clock firmware directly from a supported desktop browser.

Check your board

Only continue with the LCDWIKI ES3N28P: ESP32-S3, 16 MB flash, 8 MB PSRAM, 240x320 ILI9341 display, and ES8311 audio.

Installing this image on a different ESP32-S3 board can leave its display, storage, and audio unusable until different firmware is installed.

Install firmware

  1. Use Google Chrome or Microsoft Edge on a Windows, macOS, or Linux computer. Phones, tablets, Firefox, and Safari cannot use this installer.
  2. Connect the clock directly to the computer with a USB data cable and close any serial-monitor application.
  3. Choose Install firmware, select the ESP32-S3 serial device, and confirm the installation.
  4. Keep the cable connected until installation reaches 100% and the clock restarts.
Open this page in Chrome or Edge on a computer with Web Serial support. Browser access to the serial device was blocked. Reload the page and allow access when prompted.

Important: a normal installation preserves saved Wi-Fi, location, settings, and adhan audio. Choosing a full-device erase removes them.

Complete setup

  1. Join Adhancron Setup using password adhancron.
  2. Open 192.168.4.1, choose the home Wi-Fi network, and save its password.
  3. After the clock reconnects, open the IP address shown at the bottom of its display.
  4. Save the location, playback output, volume, enabled prayers, and adhan MP3.

After this installation, the clock checks for verified firmware updates over Wi-Fi and installs them automatically outside prayer and playback windows. USB is only needed again for recovery or a future storage-layout change.

Manual recovery

Download the merged image when browser flashing is unavailable. It must be written at offset 0x0 with esptool.
